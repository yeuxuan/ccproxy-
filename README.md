# ccproxy-一花服务器卡密管理系统

># 当前程序最新版本为 **V2.0.0**

# [**官方文档（最新）**](https://yeuxuan.github.io/ccproxy_kami/#/)

**必须使用PHP7.3-7.4 不然会有问题！！！**

**数据库必须使用Mysql5.7不然会有问题！！！**
